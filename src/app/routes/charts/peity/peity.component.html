Pie Charts
A "/" delimiter is assumed to mean "three out of five", otherwise all of the values are included in the chart and the total is the sum of all values. 
<span peity [type]="'pie'" [data]="pieData1" [options]="pieOptions"></span>
<span peity [type]="'pie'" [data]="pieData2" [options]="pieOptions"></span>
<span peity [type]="'pie'" [data]="pieData3" [options]="pieOptions"></span>
<span peity [type]="'pie'" [data]="pieData4" [options]="pieOptions"></span>
Donut Charts
Donut charts are the same as pie charts and take the same options with an added innerRadius option which defaults to half the radius. 
<span peity [type]="'donut'" [data]="donutData1" [options]="donutOptions"></span>
<span peity [type]="'donut'" [data]="donutData2" [options]="donutOptions"></span>
<span peity [type]="'donut'" [data]="donutData3" [options]="donutOptions"></span>
<span peity [type]="'donut'" [data]="donutData4" [options]="donutOptions"></span>
Line Charts
Line charts work on a comma-separated list. Line charts can take as options: delimiter, fill, height, max, min, stroke, strokeWidth and width. 
<span peity [type]"'line'" [data]="lineData1" [options]="lineOptions"></span>
<span peity [type]"'line'" [data]="lineData2" [options]="lineOptions"></span>
<span peity [type]"'line'" [data]="lineData3" [options]="lineOptions"></span>
Bar Charts
Bar charts work in the same way as line charts and take the following options: delimiter, fill, height, max, min, padding and width. 
<span peity [type]"'bar'" [data]="barData1" [options]="barOptions"></span>
<span peity [type]"'bar'" [data]="barData2" [options]="barOptions"></span>
<span peity [type]"'bar'" [data]="barData3" [options]="barOptions"></span>
Real time
Charts can be updated by changing the the jQuery selection's text content and calling change() on it. The chart will be redrawn with the same options that were originally passed to it.
